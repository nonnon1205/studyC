# 統合ロガー (Unified Logger) 詳細設計書

## 1. モジュール概要
`unified_logger` は、システム全体（各モジュール）で共通して利用されるスレッドセーフな高機能ロギング基盤です。
標準のsyslog出力に加えて、任意のファイル出力機能を備え、パフォーマンスの維持とデバッグ容易性を両立する設計となっています。

## 2. アーキテクチャと基本構造
- **Opaque Handle (不透明ポインタによる隠蔽):** 内部状態 (`struct UlogContext`) をヘッダから隠蔽し、カプセル化を実現しています (`UlogHandle`)。これにより、利用側モジュールの再コンパイルを抑えつつ、内部実装を柔軟かつ安全に変更可能です。
- **スレッドセーフティ:** マルチスレッド環境下（各種ワーカースレッドからの同時呼び出し）において、ログメッセージが混ざったりリソース競合が発生したりしないよう排他制御等を行います。

## 3. ログレベル定義 (UlogLevel)
Syslogの重要度レベルに準拠しつつ、開発段階から本番運用まで対応可能な6段階のレベルをサポートします。稼働中にMgmtモジュールから動的に出力レベル（閾値）を変更することが可能です。

- `ULOG_LEVEL_TRACE` (0): 変数値のダンプや詳細な関数のエントリ/エグジット等
- `ULOG_LEVEL_DEBUG` (1): 状態遷移や詳細な制御フローのデバッグ情報
- `ULOG_LEVEL_INFO`  (2): 正常な操作やマイルストーンの記録
- `ULOG_LEVEL_WARN`  (3): エラーではないが注意が必要な状態・閾値超過
- `ULOG_LEVEL_ERROR` (4): 介入が必要なエラー状態
- `ULOG_LEVEL_FATAL` (5): プログラムの継続が不可能な致命的エラー

## 4. 主要機能

### 4.1. 初期化と動的設定
`ulog_init(ident, level, flags)` によって初期化され、`flags` のビットマスクを用いて以下の機能を有効化できます。
- `0x01`: ファイル出力の有効化（例: `/tmp/sutdyc.log`）
- `0x02`: スレッドID (TID) のログへの付与
- `0x04`: タイムスタンプの付与（syslog以外の出力先で有用）

### 4.2. コンテキスト自動付与 (Location-Aware Macros)
開発者が都度ファイル名や行番号を記述する手間を省くため、マクロによってメタデータを自動解決し付与します。
- `ULOG_INFO(log, ...)`: 標準的なログ出力
- `ULOG_INFO_AT(log, ...)`: `__FILE__`, `__LINE__`, `__func__` を展開して `ulog_log_at()` へ渡し、障害発生箇所の特定を極めて容易にします。

### 4.3. コンテキストタグ (Context Tags)
`ulog_set_context_tag()` を用いて、処理中のリクエストIDや特定のセッション識別子を設定可能です。非同期・イベント駆動アーキテクチャにおいて、特定のリクエストがシステム内をどのように伝播したかを後からトレースする際に強力な機能となります。

### 4.4. 統計とデバッグ (Statistics)
`ulog_stats()` を通じて、処理された総ログ件数や、現在のログレベル設定によってドロップ（出力スキップ）されたログ件数を追跡します。これはMgmtモジュール経由でパフォーマンス監視のメトリクスとして利用されます。

## 5. 後方互換性 (Backward Compatibility)
既存のシステムコードからスムーズに移行するため、旧来のグローバルなログ関数（`log_init`, `log_info`, `log_warn`, `log_err`）へのラッパーインターフェースを維持しています。
これらの関数を呼び出した場合、内部ではデフォルトのグローバルロガーインスタンスへ処理がルーティングされる仕組みを採ります。

## 6. 実装上の要件 (Cファイル実装に向けて)
1. **メモリ管理**: `ulog_init` でアロケーションしたリソースは、`ulog_close` で確実に解放（メモリリークの防止）。
2. **排他制御**: ファイルポインタへの書き込みや統計情報のインクリメントは、ミューテックス（`pthread_mutex_t`等）で保護する。
3. **フォーマット**: ログ出力の際は、`vsnprintf` を使用してバッファオーバーランを安全に防止する。
